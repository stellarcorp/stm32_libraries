@par Demo Description

This is the Out-of-Box demo based on TouchGFX framework for the STM32L496G Discovery board
with 240x240 px display attached. It can be found in TouchGFX Designer as TouchGFX Demo 4.

Requirements:
     Resolution: 240 * 240
     BPP: 16

The demo features the following subdemos:
    - Menu
    - Chrom-ART demonstration
    - Weather forecast
    - Animated Graphics (Light effect)
    - Home Automation

@par Keywords

Demonstration, TouchGFX

  - This Demonstration has been tested with STM32L496G-DISCO board (MB1261-revB06).
@par How to use it ?

In order to program the demo, you must do the following:
1- Open STM32CubeProgrammer and click on "Connect"
2- Go to external loaders menu then select MX25R6435F_STM32L49G-DlSCO to be able to program the external Q-SPI flash memory
3- From "Erasing & programming" menu browse to select the STM32L496G-DISCO_OoB_v1.1.1.hex file from build folder then click on "Start Programming"
